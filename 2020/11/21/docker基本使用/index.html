  1. 创建容器后，运行shell

    • -i -t 两个选项一起使用可以分配一个终端设备然后连接到这个终端。

    • /bin/bash 表示运行容器后直接执行bash

  2. 后台运行镜像

    • -d 表示以后台的形式运行这个镜像
